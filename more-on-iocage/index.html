More on iocage

iocage is a zero dependency drop in jail/container manager, combining many of the best features and technologies the FreeBSD operating system has to offer.

It is geared for ease of use and is designed with a simplistic and easy to learn command syntax that allows you to spin up a container faster than ever before. Check out the demo below to see how iocage can spin up a Plex container all in one command.

Features

  • Templates, basejails, and normal jails
  • Easy to use
  • Rapid thin provisioning within seconds
  • Automatic package installation
  • Virtual networking stacks (vnet)
  • Shared IP based jails (non vnet)
  • Dedicated ZFS datasets inside jails
  • Transparent ZFS snapshot management
  • Binary updates
  • Export and import
  • And many more!
